# Graphic Design Basics

<span className="badge badge-beginner">Beginner</span>

- Looks professional and trustworthy
- Makes your brand memorable

## The 5 Basic Design Principles

### 1. Contrast

Make important things stand out by using differences in:
- Color (dark vs. light)
- Size (big vs. small)
- Weight (bold vs. thin)

**Example:** A bold white title on a dark background is easy to read.

### 2. Alignment

Line things up! It makes designs look clean and organized.
- Don't randomly place elements
- Use guides and grids
- Keep text aligned (left, center, or right - pick one)

### 3. Repetition

Use consistent elements throughout your brand:
- Same colors
- Same fonts
- Same style of images
- Same logo placement

This builds recognition.

### 4. Proximity

Group related things together. Space unrelated things apart.
- Title and subtitle should be close
- Different sections should have clear separation

### 5. White Space

Empty space is your friend! It:
- Makes designs easier to read
- Feels more professional
- Draws attention to what matters

<div className="fun-fact">
  <strong>Pro Tip:</strong> Beginners often try to fill every inch of space. Resist this urge! The best designs "breathe" with plenty of white space.
</div>

## Color Basics

### The Color Wheel
Colors are organized in a wheel. Understanding it helps you choose colors that work together.

### Color Combinations That Work

| Type | What It Means | Example |
|------|--------------|---------|
| Complementary | Opposite on wheel | Blue + Orange |
| Analogous | Next to each other | Blue + Purple + Pink |
| Monochromatic | Shades of one color | Light blue to dark blue |

### Choosing Brand Colors

Pick 2-4 colors for your brand:
- **Primary color** - Main brand color
- **Secondary color** - Accent color
- **Neutral colors** - Black, white, gray for text/backgrounds

### For Paws & Claws
- Primary: Warm orange (friendly, energetic)
- Secondary: Teal (trustworthy, calm)
- Neutrals: White, dark gray

### Color Psychology

Colors create feelings:

| Color | Feeling |
|-------|---------|
| Red | Energy, urgency, passion |
| Orange | Friendly, cheerful, confident |
| Yellow | Happy, optimistic, warm |
| Green | Natural, healthy, peaceful |
| Blue | Trustworthy, calm, professional |
| Purple | Creative, luxurious, mysterious |
| Pink | Playful, feminine, gentle |

## Typography (Fonts)

### Font Pairing Rules

Use 2 fonts maximum:
1. **Header font** - Bold, attention-grabbing
2. **Body font** - Easy to read

### Font Categories

| Type | When to Use | Examples |
|------|-------------|----------|
| Serif | Traditional, trustworthy | Times, Georgia |
| Sans-serif | Modern, clean | Arial, Helvetica |
| Script | Elegant, creative | Brush scripts |
| Display | Headlines only | Decorative fonts |

### Typography Tips

- Make sure text is readable
- Use size to create hierarchy (bigger = more important)
- Don't use more than 2-3 font sizes
- Ensure enough contrast between text and background

<div className="important-box">
  <strong>Readability First:</strong> A fancy font means nothing if people can't read it. Always choose clarity over creativity.
</div>

## Free Design Tools

### Canva (Recommended for Beginners)
- Free version is excellent
- Tons of templates
- Drag-and-drop interface
- Works in your browser

**Great for:** Social media posts, stories, presentations

### Other Free Options
- **Adobe Express** - Similar to Canva
- **Figma** - More advanced, but free
- **GIMP** - Free Photoshop alternative
- **Pixlr** - Quick photo editing

---

## Using Canva: Quick Start

1. **Go to** canva.com and create free account
2. **Search** for template (e.g., "Instagram post")
3. **Choose** a template that fits your style
4. **Customize:**
   - Change text
   - Swap images
   - Adjust colors
   - Move elements
5. **Download** as PNG or JPG

### Canva Tips
- Start with templates, then customize
- Use their stock photos (millions available free)
- Keep text minimal on graphics
- Use brand kit to save your colors

## Image Best Practices

### Size Matters

Each platform has ideal sizes:

| Platform | Post Size | Story Size |
|----------|-----------|------------|
| Instagram | 1080x1080 (square) | 1080x1920 |
| TikTok | 1080x1920 | Same |
| Facebook | 1200x630 | 1080x1920 |
| Twitter/X | 1200x675 | N/A |

### Image Quality

- Use high-resolution images
- Avoid blurry or pixelated photos
- Make sure images aren't stretched

### Stock Photos

Free stock photo sites:
- **Unsplash** - High quality, free
- **Pexels** - Large library
- **Pixabay** - Also has videos
- **Canva** - Built into the tool

## Creating Consistent Branding

Make all your content look like it belongs together:

### Brand Consistency Checklist
- [ ] Same logo placement
- [ ] Same colors
- [ ] Same fonts
- [ ] Same style of photos
- [ ] Same overall vibe

<div className="tip-box">
  <strong>The Instagram Test:</strong> Look at your last 9 posts as a grid. Do they look like they belong together? That's good branding.
</div>

## Common Design Mistakes

| Mistake | Fix |
|---------|-----|
| Too many fonts | Stick to 2 |
| Too many colors | Use 3-4 max |
| Hard to read text | Increase contrast, size |
| Cluttered design | Add white space |
| Low-quality images | Use HD photos |
| Inconsistent style | Create brand guidelines |
